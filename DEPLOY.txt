Locket Dio on Render — CORS proxy fix

Problem:
  api.locket-dio.com only allows Origin: https://locket-dio.com (and localhost).
  Custom domain locket-dio-ly9t.onrender.com gets "Not allowed by CORS" → Server lỗi.

Fix:
  Node server.mjs serves public/ and proxies:
    /dio-api/*      → https://api.locket-dio.com/*
    /dio-data/*     → https://data.locket-dio.com/*
    /dio-storage/*  → https://storage.locket-dio.com/*
  with Origin spoofed to https://locket-dio.com

IMPORTANT — service type must be WEB (Node), not Static:
  1. Render Dashboard → New → Web Service
  2. Repo: buiduchuy2010qn-prog/locket-dio
  3. Runtime: Node
  4. Build: echo "Using prebuilt public/ + server.mjs proxy"
  5. Start: node server.mjs
  6. Or open existing service Settings → if Static, create NEW Web Service instead

URL: https://locket-dio-ly9t.onrender.com  (or new URL if recreated)
