# Piclet Gold

Private close-friends photo sharing web app — original branding inspired by the *concept* of intimate moment sharing (not affiliated with Locket or any third-party brand).

## Stack

- React 19 + Vite 8
- Tailwind CSS v4
- React Router 7
- Lucide icons
- Mock API + localStorage (backend-ready placeholders)

## Quick start

```bash
cd piclet-gold
npm install
npm run dev
```

Open http://localhost:5173

## Features

- Landing, auth (login / signup / forgot)
- Feed, upload (camera + roll Gold lock), friends, requests
- Notifications, gallery, streaks (restore Gold)
- Profile, settings, dark mode
- Full Piclet Gold: pricing, comparison, customization (icons, camera themes, badges, frames)
- Admin mock debug page
